# Images Folder

## How to Add Images:

1. **Copy your image files** into this folder (`public/images/`)
2. **In the admin panel**, use the image filename like: `/images/your-image.jpg`
3. **Push to GitHub** and Vercel will automatically deploy

## Example:
- If you add `service1.jpg` to this folder
- In admin panel, use: `/images/service1.jpg`

## Supported Formats:
- JPG/JPEG
- PNG
- WebP
- SVG

## Tips:
- Keep image file sizes under 500KB for fast loading
- Use descriptive names like `academic-support.jpg`
- Optimize images before uploading (use tinypng.com)
